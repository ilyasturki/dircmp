# dircmp

Terminal TUI for comparing two directories side by side.

## Stack

- TypeScript, React, Ink (TUI framework)
- Package manager: bun

## Commands

- `bun run dev <left-dir> <right-dir>` — run the app
- `bun run build` — compile to standalone binary (`dircmp`)
- `bun test` — run tests
- `bun run typecheck` — type-check
